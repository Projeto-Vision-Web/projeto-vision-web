# VISION-WEB

Aplicação composta por:

- **Backend**: API em Java / Spring Boot  
- **Frontend**: app web (Vue/Vite)  
- **Banco de dados**: MySQL 8 em container  

Todo o ambiente de desenvolvimento pode ser iniciado com **Docker Compose**.

---

## ✅ Pré-requisitos

Antes de começar, o dev precisa ter instalado:

- [Docker Desktop](https://www.docker.com/products/docker-desktop) (Windows / Mac)  
- Ou Docker Engine + Docker Compose (Linux)

Para testar se está tudo ok:

```bash
docker --version
docker compose version
